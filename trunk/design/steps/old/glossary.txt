
Glossary
========

Function 

A C/C++ function including member fucntions. Not used to refer to HDL funcitons (we choose not to use these).

Module

A hierarchy element similar too verilog's modules

Client/server

Clients and servers are modules such that the client requests a service from a server via a protocol and waits for a result. This process is called a transaction.

Transaction

See client/server
 
Concurrancy

In this work, we take concurrancy to mean the presence of more than one cause-effect chain in the system. When there is only one we regard the system as parallel but not concurrent since it is emulating a non-parallel system.
